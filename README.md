## Overview
Geoclaw is an autonomous AI agent framework designed for Business Intelligence (BI) and Open Source Intelligence (OSINT).

## Quick Start
1. **Install:**
   - Mac/Linux: `./install.sh`
   - Windows: Double-click `install.bat`
2. **Configure:** Run `python configure.py`
3. **Run:** Start the dashboard with `python tui.py`
